## find 查找文件或者目录 查找文件或者目录
选项 功能
-name<查询方式> 按照指定的文件名查找模式查找文件
-user<用户名> 查找属于指定用户名所有文件
-size<文件大小> 按照指定的文件大小查找文件,单位为:
b —— 块（512 字节）
c —— 字节
w —— 字（2 字节）
k —— 千字节
M —— 兆字节
G —— 吉字节

 find xiyou/ -name "*.txt"
 find xiyou/ -user gcy
 ## 查找大于200m的文件（+n 大于 -n小于 n等于）
 find /home -size +204800

              # 3.grep 和 | 常用方法

              ## 1、从单个文件中查找指定的字符串
# grep "搜索字符" 文件名
grep "this" demo_file
## 2、从多个文件中查找指定的字符串
grep "this" *.txt
## 3、查询且忽略大小写
grep -i
## 4、递归搜索所有的文件及子目录
grep -r
## 5 -c 统计字符出现的次数
grep -c
## 6 -n 输出行号
grep -n
#7 -v 反向匹配,从末尾开始找
grep -v 经常 -vn
#8 配合|使用
如
ls | grep target
ls | gerp "string"
#9 查找多个字符串
#您也可以使用grep查找多个单词或字符串。您可以使用-e开关指定多个模式
grep -e 'Class 1' -e Todd Students.txt


#10 显示之前和之后的行
#如果需要更多grep输出上下文，可以使用-c开关在指定的搜索字符串前后显示一行：
grep -c 1 string filename
# 结果显示了 3行 包括了前后各一行

# 11对结果进行排序
#将grep的输出传递给sort命令，以按某种顺序对结果进行排序。默认为字母顺序。
 grep string filename | sort

              ## grep配合正则表达式
# grep '正则' 文件  要注意转义
grep '[0-9]\{1,3\}' test.txt
